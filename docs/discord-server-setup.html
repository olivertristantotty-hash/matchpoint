1. Create the Server

1 Open Discord → click the + button on the left sidebar → Create My Own.
2 Select "For a club or community".
3 Name it something clean and memorable. Examples: WagerZone, StakeUp, The Arena.
4 Upload a server icon (square, at least 512×512px).
5 Go to Server Settings → Community and enable Community Server. This unlocks the welcome screen, server discovery, and announcement channels.

Tip: Don't gate wagering behind any role. Let anyone use /wager immediately. Roles are for badges and trust signals, not access control.

4. Bot Installation

1 Go to discord.com/developers/applications and select your WagerBot application.
2 Go to OAuth2 → URL Generator.
3 Under Scopes, check: bot and applications.commands.
4 Under Bot Permissions, check:
  • Send Messages
  • Send Messages in Threads
  • Embed Links
  • Attach Files
  • Read Message History
  • Use Slash Commands
  • Add Reactions
5 Copy the generated URL, open it in your browser, and select your server.
6 In Server Settings → Integrations, verify WagerBot appears and its commands are enabled.
Important: In the Discord Developer Portal under Bot → Privileged Gateway Intents, enable Server Members Intent. This is required for the welcome DM feature.

5. Welcome Message & Rules

#welcome — Post This Exactly

Welcome to [Server Name]! 🎮

Challenge anyone to a match. Stake tokens. Winner takes the pot.

Get started in 30 seconds:
→ /deposit 1000 — grab free starter tokens
→ /wager @opponent game amount — challenge someone
→ /link — connect your game accounts for auto-verified results

Head to #find-match to find an opponent. Good luck.

#rules — Post This

Rules

1. No fake results. Submitting a fake screenshot or lying about a result = strike. 3 strikes = permanent ban.

2. Submit screenshots. After every match, both players submit a screenshot of the final score using /submit. The system reads it automatically.

3. Respect deadlines. You have 90 minutes to play and submit after a wager is accepted. No-shows get a reputation penalty.

4. No collusion. Alternating wins with the same person to farm tokens will get both accounts banned.

5. Disputes are public. If there's a disagreement, it goes to #disputes. Mods review evidence and decide. Their call is final.

6. Be cool. Trash talk is fine. Harassment is not.

#link-accounts — Post This

Link Your Game Accounts

Linking lets the bot verify your match results automatically — no screenshots needed for supported games.

/link platform:Riot username:YourName#TAG — League of Legends & Valorant
/link platform:EA username:YourEAID — FIFA / EA FC
/link platform:Steam username:YourSteamID — Rocket League
/link platform:Epic username:YourEpicName — Fortnite
/link platform:Activision username:YourActiID — Call of Duty

Auto-verified games: League of Legends, Valorant (via Riot API)
Screenshot-verified: FIFA, Rocket League, CoD, Fortnite

You can wager without linking — you'll just need to submit screenshots manually.

7. Moderation Setup

What Moderators Do

  • Review disputes in #disputes
  • Look at evidence in #evidence
  • Use /resolve dispute_id outcome to settle disputes
  • Outcomes: creator_wins, opponent_wins, or refund

Moderator Guidelines

  1. Always check both players' screenshots before deciding.
  2. If one player submitted a screenshot and the other didn't, the player with evidence wins.
  3. If both screenshots are unclear, refund both players.
  4. If a screenshot is clearly edited/faked, the other player wins and the faker gets a strike via the reputation system.
  5. Be fast — disputes should be resolved within 2 hours during active hours.

AutoMod Recommendations

In Server Settings → AutoMod, set up these rules:

  • Block common slurs and harassment
  • Block spam (5+ identical messages in 10 seconds)
  • Block links in #active-wagers and #results (bot-only channels)
